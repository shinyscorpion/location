# Location

Location and language database.

## Installation

The package can be installed
by adding `location` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:location, "~> 0.0.1"}
  ]
end
```

Tthe docs can
be found at [https://hexdocs.pm/location](https://hexdocs.pm/location).
